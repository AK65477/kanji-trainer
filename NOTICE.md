# Notices and Data Sources

Kanji Trainer is an Android app for practicing Japanese kanji readings in short
sentence contexts.

## Project Code

The app source code is licensed under the MIT License. See `LICENSE`.

## Bundled Study Data

The bundled seed file is:

- `app/src/main/assets/seed/kanji_seed.json`

It contains kanji metadata and sentence-card study material curated for this app.
For public distribution, the seed data should be treated conservatively as a
separate data work from the app code.

To keep redistribution simple and compatible with common Japanese dictionary
reference sources, the bundled study data is distributed under:

- Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0)
- https://creativecommons.org/licenses/by-sa/4.0/

Where the seed data includes original project-authored example sentences,
distractors, Korean keywords, or card structure, those app-specific data
contributions are also made available under CC BY-SA 4.0.

## Reference Sources

The seed data may be checked against or informed by public Japanese language
reference resources, especially for kanji readings, classification, and visual
component/decomposition information.

Relevant sources include:

- Electronic Dictionary Research and Development Group (EDRDG)
- KANJIDIC2: kanji readings and metadata
- RADKFILE/KRADFILE: visible kanji components
- JMdict/EDICT project documentation

EDRDG licence statement:

- https://www.edrdg.org/edrdg/licence.html

KANJIDIC project:

- https://www.edrdg.org/wiki/index.php/KANJIDIC_Project

JMdict/EDICT project:

- https://www.edrdg.org/wiki/index.php/JMdict-EDICT_Dictionary_Project

EDRDG dictionary files are made available under Creative Commons
Attribution-ShareAlike 4.0. This app does not imply endorsement by EDRDG or its
contributors.
