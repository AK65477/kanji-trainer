# Security Policy

## Supported Versions

Kanji Trainer is currently a small public MVP. Security fixes apply to the latest release only.

## Reporting A Vulnerability

Please open a GitHub issue if you find a security or privacy problem that does not expose anyone's private information.

If the report includes sensitive details, contact the maintainer privately through GitHub profile contact methods instead of posting secrets in a public issue.

## Privacy Model

Kanji Trainer is local-first. It does not include analytics, ads, accounts, network sync, or third-party tracking.
